# Personal_Finane_Tracker1
A simple and modern Personal Finance Tracker built using:
- HTML
- CSS
- JavaScript
- Local Storage
- Free Quote API
This project helps users manage their income and expenses directly in the browser.
Features
Add Income & Expense
Delete Transactions
Clear All Transactions
Calculate Total Balance
Calculate Total Income
Calculate Total Expense
Data Stored in Local Storage
Responsive UI Design
Finance Quote API Integration
Technologies Used:
Frontend:
- HTML5
- CSS3
- JavaScript (Vanilla JS)
API Used:
txt
https://dummyjson.com/quotes/random
